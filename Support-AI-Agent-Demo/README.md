# Support AI Agent (Demo Version)

This is a template/demo version of an AI-powered technical support agent, designed for portfolio and educational purposes. **No real or sensitive data is included.**

## Features
- Hybrid retrieval (dense + sparse) for support ticket search
- LLM-based answer generation (plug in your own API keys/models)
- Modular, scalable, and multilingual-ready
- CLI demo interface

## How to Use
1. Clone the repo
2. Install dependencies:  
   `pip install -r requirements.txt`
3. Add your own LLM API keys and endpoints in `config_sample.json`
4. Run the CLI demo:  
   `python app/cli_app.py`

## Sample Data
The `data/sample_tickets.json` file contains a few example support tickets for demonstration.

## Disclaimer
This repository is a public demo and does not contain any proprietary or sensitive information. For real-world use, integrate with your own data and infrastructure.
